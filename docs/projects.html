Enlace Github

Ejercicio de evaluación intermedia que desarrollamos durante el examen.

Las Retamas

Enlace Github

Página web del conocido grupo de música alcorconero Las Retamas.
Página web pensada para darle más visibilidad a esta banda de música de Alcorcón. En principio está pensada para anunciar las noticias de la banda, pero con el pensamiento de evolucionarla y que sea tienda online para vender merchandising.

SerioFinder

Enlace Github

Ejercicio de evaluación de JavaScript
El proyecto consiste en pedir los datos a una API, y mostrar el listado de series. Por otro lado te deja filtrar según el nombre del input y elegir tu serie favorita, dejándote marcarla y desmarcarla.

Botanical Profile Cards

Enlace Github

Proyecto de equipo utilizando React.

Rick y Morty

Enlace Github

Ejercicio de evaluación de React.
El proyecto consiste en un buscador de personajes de la serie Rick y Morty. Te permite filtrar pro nombre y por especie. También ver en detalle la información de cada personaje.
